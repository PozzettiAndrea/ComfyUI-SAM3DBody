# Example Assets

This folder contains example assets for testing SAM 3D Body:

## Files

- `model_diagram.png`: SAM 3D Body model architecture diagram
- Example images from the SAM 3D Body qualitative comparisons

## Usage

Use these images in ComfyUI to test the SAM 3D Body nodes:
1. Load any of the example images with the "Load Image" node
2. Connect to "Load SAM 3D Body Model" and "SAM 3D Body: Process Image"
3. Visualize the results
